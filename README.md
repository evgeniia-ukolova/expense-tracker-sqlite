# Expense Tracker SQLite

Учебное консольное приложение для учёта расходов на Python с использованием SQLite.

Проект позволяет добавлять расходы, хранить их в локальной базе данных, просматривать статистику по категориям и месяцам, искать расходы и редактировать записи.


## Статус проекта

Проект создан как учебный CLI-трекер расходов для практики Python, SQLite, SQL и Git.

Цель проекта — не продакшен-разработка, а закрепление базовых навыков: работа с функциями, разделение кода по файлам, хранение данных в SQLite, оформление README и публикация проекта на GitHub.


## Возможности

- добавление расхода;
- просмотр всех расходов;
- удаление расхода по ID;
- редактирование расхода;
- подсчёт общей суммы расходов;
- фильтр расходов по категории;
- подсчёт суммы по категории;
- фильтр расходов по месяцу;
- подсчёт суммы за месяц;
- подсчёт количества расходов за месяц;
- подсчёт среднего расхода за месяц;
- поиск самого большого расхода за месяц;
- поиск самого маленького расхода за месяц;
- поиск расхода по названию;
- поиск расхода по описанию;
- просмотр списка категорий;
- автоматическая дата сегодняшнего дня, если дата не введена;
- сохранение данных в SQLite.

## Пример расхода

ID: 4 | 08.07.2026 | кафе | кекс — 200.00 | десерт после обеда 

Поля расхода:

- ID — уникальный номер записи;
- date — дата расхода;
- category — категория;
- title — название расхода;
- amount — сумма;
- description — описание или комментарий.

## Особенности реализации

- данные хранятся в локальной SQLite-базе expenses.db;
- таблица создаётся автоматически при запуске приложения;
- для добавления новой колонки description используется миграция через ALTER TABLE;
- SQL-запросы используют безопасную подстановку значений через ?;
- дата хранится в базе в формате YYYY-MM-DD;
- пользователю дата выводится в формате DD.MM.YYYY;
- категории приводятся к нижнему регистру через .lower();
- дата может подставляться автоматически, если пользователь нажал Enter;
- для поиска используется SQL LIKE;
- для аналитики используются SUM, COUNT, AVG, GROUP BY, ORDER BY, LIMIT.

## Примеры SQL-запросов из проекта

Добавление расхода:

sql INSERT INTO expenses (title, amount, category, date, description) VALUES (?, ?, ?, ?, ?) 

Получение всех расходов:

sql SELECT id, title, amount, category, date, description FROM expenses ORDER BY date DESC, id DESC 

Подсчёт общей суммы:

sql SELECT SUM(amount) FROM expenses 

Суммы по категориям:

sql SELECT category, SUM(amount) FROM expenses GROUP BY category ORDER BY SUM(amount) DESC 

Суммы по месяцам:

sql SELECT SUBSTR(date, 1, 7), SUM(amount) FROM expenses GROUP BY SUBSTR(date, 1, 7) ORDER BY SUBSTR(date, 1, 7) DESC
